Introduction
============

The Referentielijsten API is a generic API for 'stamtabellen' that contain data that rarely changes.
The data can be used as reference or to support relations with other tables in a database.

A `stamtabel`, `referentietabel` or lookup-tabel` can for for example contain:
country codes, street names, product categories or lists of static values such as colors or sizes.
