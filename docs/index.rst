Documentation
=============

**Referentielijsten** is an application where tables and items can be
managed. Other applications like `Open Formulieren`_ can integrate **Referentielijsten** using
its REST API to for example fill a radio components with items of a list.

.. _`Open Formulieren`: https://open-forms.readthedocs.io/nl/latest/


This component is designed in line with the `Common Ground`_ model.

This project is and only uses :ref:`introduction_open-source`.

.. _`Common Ground`: https://commonground.nl/
